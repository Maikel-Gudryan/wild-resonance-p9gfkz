# VAPOReon 🎮

Projeto desenvolvido com React + Vite.

## Sobre

VAPOReon é uma interface de loja de jogos inspirada na Steam, com cards dinâmicos, categorias e visual moderno.

## Tecnologias

- React
- Vite
- JavaScript
- CSS

## Funcionalidades

- Lista de jogos
- Cards interativos
- Sistema de categorias
- Página de detalhes
- Interface responsiva
